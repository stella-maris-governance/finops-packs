# Screenshot Capture Guide — Reserved Instance Optimization

---

### 01 — Azure Advisor Recommendations
**Path:** Azure Portal > Advisor > Cost > Reservation Recommendations
**Show:** List of RI recommendations with estimated savings

### 02 — Fitness Score Output
**Path:** `reservation-fitness-score.py` terminal output
**Show:** Scored candidates with per-factor breakdown

### 03 — Break-Even Analysis
**Path:** `break-even-calculator.py` output
**Show:** Per-candidate break-even with risk scenarios

### 04 — Utilization Dashboard
**Path:** Azure Cost Management > Reservations > Utilization
**Show:** Reservation utilization percentage (post-purchase)

### 05 — Savings Report
**Path:** `savings-tracker.json` or Cost Management savings view
**Show:** Actual vs on-demand counterfactual
